Exoflex

UI component library for React Native and Web (using react-native-web).

DocsDemo

Easy to Use

Almost no setup required. Can even be used without Provider.

Almost Core UI Free

By using the Provider you could add custom style for every component on the theme so there is no need to create another file for custom core-ui.

Multi-platform Out of the Box

You could use it either on native app or web app as long you are using react-native-web on your project.
